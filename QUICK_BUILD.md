# ⚡ 快速打包指南

## 🎯 目标：5 分钟创建可执行文件

---

## 📋 打包步骤

### 第 1 步：一键打包

**macOS/Linux:**
```bash
./build.sh
```

**Windows:**
```cmd
build.bat
```

就这么简单！脚本会自动：
- ✅ 检查并激活虚拟环境
- ✅ 安装 PyInstaller
- ✅ 清理旧文件
- ✅ 执行打包
- ✅ 显示结果

---

## 📦 打包结果

打包完成后，可执行文件在：

```
dist/YOLOv8训练系统/
├── YOLOv8训练系统      # 可执行文件（macOS/Linux）
├── YOLOv8训练系统.exe  # 可执行文件（Windows）
└── _internal/         # 依赖文件
```

---

## 🚀 使用方法

### 开发者（你）：
1. 运行 `./build.sh` 打包
2. 将 `dist/YOLOv8训练系统/` 文件夹打包成 zip
3. 发送给用户

### 用户（接收者）：
1. 解压 zip 文件
2. 双击运行 `YOLOv8训练系统`
3. 浏览器自动打开
4. 开始使用！

---

## ❓ 用户需要配置环境吗？

### ❌ 完全不需要！

用户只需要：
- ✅ 一台电脑
- ✅ 解压软件
- ✅ 双击能力

**不需要：**
- ❌ 安装 Python
- ❌ 安装依赖
- ❌ 配置环境变量
- ❌ 任何编程知识

这就是打包的意义！🎉

```bash
# 1. 打包

```

## 🐛 遇到问题？

### 打包失败？
1. 检查虚拟环境是否存在
2. 确保所有依赖已安装：`pip install -r requirements.txt`
3. 查看错误信息

### 运行失败？
1. 检查是否完整解压
2. 在控制台窗口查看错误信息
3. 确保端口 7865 未被占用
